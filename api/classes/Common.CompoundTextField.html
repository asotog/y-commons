Defined in: gallery-y-common-compound-textfield/js/gallery-y-common-compound-textfield.js:1
Module: gallery-y-common-compound-textfield

This module is a widget that syncs multiple fields into one field usually a hidden field to store a compound value for example a social security number or phone number where usually needs like 3 different fields
