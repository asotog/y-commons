The following example shows how to make work the compounds fields using separator

Hidden value:

Basic Example

This component depends on the inputs provided using the repective classes that also can be easily modified, please check api docs for further details.

If hidden field is already set the component will take its value and will set each of the compound fields:

HTML

<!-- component include  -->
<script src="/build/gallery-y-common-compound-textfield/gallery-y-common-compound-textfield.js"></script>

<div id="phone-container1">
    <input type="text" class="compound-text-field" maxlength="3">
    <input type="text" class="compound-text-field" maxlength="3">
    <input type="text" class="compound-text-field" maxlength="4">
    <input type="hidden" class="main-compound-field" value="">
</div>

Javascript

Using separator attribute will split each value by using a sign:

YUI().use('gallery-y-common-compound-textfield', function(Y) {
    var compound = new Y.Common.CompoundTextField({
        srcNode: '#phone-container1',
        separator: '-'
    }).render();
    
    
    /* optional: just to show how the hiddent value is reflected */
    var output = Y.one('#phone-container1 .output span');
    
    // optional: refresh output
    compound.on('compoundValueChange', function(e) {
        output.set('text', e.newVal);
    });

});
