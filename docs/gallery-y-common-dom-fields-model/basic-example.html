Basic Example

This example will show how this component can retrieve the set of fields values and convert them into javascript object literal, also the load button will refill the fields after save and then reset button is clicked.

If save button is clicked, the object result can be seen and how the fields are retrieved.

HTML

This markup is a regular form, each field name attribute will be used as the key of the model fields, also checkbox value will be true or false.

Javascript

Javascript part is simple, only need to provide the container (it can be the form node or any other dom element wrapping the fields) and component will do the rest, also as an example, the model is stringified to show the result object:

YUI().use('gallery-y-common-dom-fields-model', 'json-stringify', function(Y) {
        
    var containerNode = Y.one('#registration-model');
    var registrationModel = new Y.Common.DomFieldsModel({
        container: containerNode
    });
    containerNode.one('.save').on('click', function() {
        registrationModel.save();
        var model = registrationModel.retrieveModel();
        alert(Y.JSON.stringify(model));
    });
    
    containerNode.one('.load').on('click', function() {
        registrationModel.load();
    });
    

});
